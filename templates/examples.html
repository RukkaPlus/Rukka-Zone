    Matematicas

    Para el uso de matematicas avanzadas, podemos usar el modulo que nos ofrece la API de Python math
    Este modulo contiene varias de las funciones mas utilizadas en las matematicas; el modulo, al pertenecer por defecto en la API de Python, puede ser importada globalmente

    import math

variable = 3.1415
# Para redondear un numero (esta funcion, viene autoincorporada
# en versiones actuales)
round(variable)

# Para redondear un numero a la baja
math.floor(variable) # -> 3

# Para redondear un numero al alza
math.ceil(variable) # -> 4

# Para saber el valor absoluto de un numero (Por defecto)
abs(-10) # -> 10

# Para realizar una suma de varios elementos de una lista (Por defecto)
sum([1, 2, 3, 4, 5, 6, 7, 8, 9]) # -> 45

# Para realizar una suma de varios elementos de una lista (Mejorada por Math)
math.fsum([0.9999999999, 1, 2, 3]) # -> 6.9999999999

# Para realizar un truncamiento
math.trunc(100.123456) # -> 100

# Para realizar una potencia (Por Math)
math.pow(2, 3) # -> 6

# Para buscar la raiz cuadrada (Por math)
math.sqrt(9) # -> 3

# La constante PI
math.pi # -> 3.141592653589793

# La constante de Euler
math.e # -> 2.718281828459045

    Numeros aleatorios

    En Python tenemos un modulo bastante interesante que permite la generacion de numeros aleatorios de manera sencilla, ese modulo es random

    import random

# Para obtener un valor aleatorio
random.random() # Esto seria un valor >= 0 y < 1.0

# Para obtener un valor aleatorio entre dos definidos
random.uniform(1, 10) # Entre 1 y 10

# Para sacar un valor aleatorio hasta uno definido
random.randrange(10) # Entre 0 y 9

# Para sacar un numero aleatorio entre dos definidos
random.randrange(0, 101, 2) # Entre 0 y 100 pero solo contando los multiplos de 2

# Para sacar un item de una coleccion
string = "Hello, world!"
lista = ["uno", "dos", "tres"]
random.choice(string)
random.choice(lista)

# Para abarajar aleatoriamente y exparcir los items de una lista
random.shuffle(lista)

# Para tomar una muestra aleatoria de una lista o coleccion
random.sample(lista, 2) # Tomara dos pruebas aleatorias

    Horarios

    Para trabajar con informacion horaria, el modulo datetime de Python nos facilita mucho una gran cantidad de trabajos referentes a ello

    import datetime

# Para obtener el tiempo actual
tiempo_actual = datetime.datetime.now

# Para obtener un tiempo personalizado
tiempo_personalizado = datetime.datetime(2000, 1, 1, 0, 0)

# Para para reemplazar un tiempo personalizado
tiempo_personalizado.replace(year=3000, moth=2, day=15, hour=19, minute=45, second=28, microsecond=607208)

# Para sumar o restar tiempos
tiempo_de_distancia = datetime.timedelta(days=14, hour=4)
dentro_de_dos_semanas = tiempo_actual + tiempo_de_distancia
hace_dos_semanas = tiempo_actual - tiempo_de_distancia

# Para formatear de forma estandar el tiempo
tiempo_actual.isoformat()

# Para formatear de forma personalizada el tiempo
tiempo_actual.strftime("%A %d %B %Y %I:%M")
# El dia              %A
# El dia de la semana %d
# El mes (Escrito)    %B
# El anio             %Y
# La hora             %I
# Los minutos         %M

    Rutas de archivos/directorios

    Los sistemas operativos pueden tener diferentes sistemas de enrutamiento a archivos o directorios, como es el caso de MSWindows y los sistemas basados en UNIX, que poseen el separador direccion\del\archivo_directorio y direccion/del/archivo_directorio respectivamente
    Para mejorar la portabilidad de una aplicacion, podemos usar la funcion join de os.path

    from os.path import join
path = join("my", "directory", "filename.txt") # -> 'my\directory\filename.txt'
                                               # -> 'my/directory/filename.txt'
file = open(path, "a+")
file.write("Hello, world!")
file.close()

if name in ('nt', 'dos'):  # If Machine is running on Windows...

    Pickle

    Nota: Pickle no es totalmente seguro, y durante el desamblaje puede ejecutar codigo malicioso no deseado, por lo que es recomendable emplear sistemas de notacion no cifrados mas seguros como JSON, YAML, TOML, etc
    Aunque Pickle puede ser considerada una buena opcion para la encriptacion de datos en sistemas locales (claro, contando con opciones seguras dentro de ellas)

    La libreria Pickle es una libreria que puede llegar a ser muy util a la hora de manejar archivos binarios ensamblados. Ademas de poseer caracteristicas bastante interesantes como la encriptacion de datos, la compresion y la carga de submodulos encriptados directamente

    lista = [1, 2, 3, 3, 4, 5, 6, 7, 8]

# [ Escribiendo en binario ]
# Abrimos un archivo en modo escritura binaria ('wb')
archivo = open('archivo.pckl', 'wb') # La extension puede ser cualquiera

pickle.dump(lista, archivo) # Primero indicamos el item y despues el archivo
archivo.close()             # Con esto debemos cerrar el archivo
# Nota: El archivo se escribira en binario, por lo que sera
# ilegible a nuestros ojos

# [ Lectura en binario ]
# Abrimos un archivo en modo lectura binaria ('rb')
archivo = open('archivo.pckl', 'rb')

# Y para recargar el fichero en lista, podemos hacer lo siguiente
lista_recup = pickle.load(archivo)

print(lista_recup)

    JSON

    JSON (JavaScript Object Notation) es un formato de archivo que permite almacenar datos en notacion de Objetos de JavaScript. Esto facilitando la lectura, la escritura y el manejo en general del archivo

    {
  "name": "PC Game",
  "default": {
    "profile_name": "Profile",
    "player_name": "Player",
    "profile_created": "1979-05-27T15:32:00.000Z",
    "sound": true
  },
  "user": {
    "profile": {
      "name": "profile1",
      "player_name": "player1",
      "class": "warrior"
    },
    "settings": {
      "sound": false
    }
  },
  "default_inventory": {
    "warrior": [
      ["sword", "shield"],
      ["plate armor", "plate helm"]
    ],
    "mage": [
      ["staff", "wand"],
      ["robe", "hood"]
    ],
    "items": [
      "health potion",
      "antidote",
      "mana potion"
    ]
  }
}

    El modulo json de Python permite trabajar con archivos en formato JSON de manera bastante sencilla. A continuacion, algunos ejemplos

    import json

# Convertir JSON a 'dict'
my_json = '{ "name": "RukkaPlus", "lang": "Python" }'
my_dict = json.loads(my_json)
type(my_dict) # -> 'dict'

# Convertir 'dict' a JSON
my_dict = { "name": "RukkaPlus", "lang": "Python" }
my_json = json.dumps(my_dict)
type(my_json) # -> 'str'

    Reglas para el uso de JSON

    Existen ciertos parametros que merecen ser conocidos al trabajar con archivos de formato JSON

    • Siempre se actua como con un Objeto de JavaScript (siempre se usa la sintaxis {"nombre de llave" : "valor"})
    • Solo se permiten valores de tipo Number, String, Array, Object, Boolean y null. Esta prohibido otros tipos de datos (incluyendo Function, Date y undefined)
    • Se mantiene la sintaxis original de agrupacion de datos (los [] representan Arrays y los {} representan Objetos)

    TOML

    name = "PC Game"

[default]
profile_name = "Profile"
player_name = "Player"
profile_created = 1979-05-27T07:32:00-08:00
sound = true

[user]

  [user.profile]
  name = "profile1"
  player_name = "player1"
  class = "warrior"

  [user.settings]
  sound = false

[default_inventory]
warrior = [ ["sword", "shield"], ["plate armor", "plate helm"] ]
mage = [ ["staff", "wand"], ["robe", "hood"] ]
items = [
  "health potion",
  "antidote",
  "mana potion"
]

    $ python3 -m pip install toml

    pyinstaller

    Se trata de una herramienta que permite cerrar el codigo de una aplicacion o programa desarrollado en Python. De esta forma podremos comercializarla y portarla facilmente, ya que junto a la compilacion de la aplicacion o programa, tambien compila el mismo interprete. De esta forma evita que sea requerido instalarlo para la ejecucion de la aplicacion

    Instalacion

    Para poder utilizarlo, primero es necesario instalar esta utilidad en el sistema, lo que es sencillo usando el gestor de paquetes integrado pip

    > py -m pip install pyinstaller
    $ python3 -m pip install pyinstaller

    Esto instalara la version estable de la utilidad mencionada

    Utilizacion

    Su uso es mediante la ejecucion de un archivo binario llamado pyinstaller. Puede ejecutarlo desde una terminal o linea de comandos

    Para ver todos los comandos disponibles puede ejecutar el siguiente comando

    > pyinstaller -h

    Para cerrar el codigo de un programa o aplicacion, ejecute el siguiente comando

    > pyinstaller direccion\del\script.py

    Esto retornara algunas carpetas y archivos relacionados a la aplicacion o programa compilado. Entre ellas, se encuentra el programa en formato binario

    Para ver todos los comandos disponibles puede ejecutar el siguiente comando

    $ pyinstaller -h

    Para cerrar el codigo de un programa o aplicacion, ejecute el siguiente comando

    $ pyinstaller direccion/del/script.py

    Esto retornara algunas carpetas y archivos relacionados a la aplicacion o programa compilado. Entre ellas, se encuentra el programa en formato binario

    Nota: Cabe recalcar que pyinstaller NO es multiplataforma y que TAMPOCO puede compilar un programa o aplicacion para diferentes sistemas desde uno solo. Lo que quiere decir que si desea compilar un programa o aplicacion para diferentes sistemas, debera compilarlo en cada sistema respectivamente
