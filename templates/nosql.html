Base de datos NoSQL

Una base de datos es un contenedor de informacion, donde podemos guardar y almacenar datos referentes a un tema en especifico. Las bases de datos pueden dividirse entre Relacionales y No relacionales. En este caso, las bases de datos no relacionales, a diferencia de las relacionales, buscan mayor velocidad que organizacion, puesto a que en estos es admitible la repeticion de datos (esto es compensado por las altas velocidades de operacion que ofrecen). En estos sistemas de bases de datos, por lo general, no se utiliza la tecnologia SQL, y en su lugar, ofrecen una API propia que permite el acceso al CRUD.
Algunos ejemplos famosos de este tipo de sistemas gestores bases de datos son MongoDB, Redis, Cassandra, CouchDB, etc

Una base de datos, por otro lado, es un conjunto de datos serializados que funcionan en formato de tablas con campos y registros.
