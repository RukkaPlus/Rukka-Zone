HTML

HyperText Markup Language (HTML) es el lenguaje Web para definir la estructura de una pagina Web
Su syntax es muy sencilla de entender, ya que este lenguaje solo sirve para estructurar paginas Web

<!-- Comentarios. Estos son ignorados por el navegador -->

Etiqueta html

La etiqueta mas fundamental de un archivo HTML es la etiqueta de mismo nombre html. Dentro de esta etiqueta se coloca todo el codigo HTML que compone a la pagina

<!DOCTYPE html> <!-- Etiqueta que indica el uso de HTML:5 -->
<html>  <!-- Inicio de la etiqueta -->
</html> <!-- Final de la etiqueta -->

Etiqueta head

La etiqueta de cabeza head es importante para definir los datos mas abstractos de un documento. Por ejemplo, el titulo de la pagina, el icono, las integraciones, los metadatos (descripciones, keywords, etc)

<!DOCTYPE html>
<html>
<head> <!-- Inicio de la etiqueta -->
  <title>Documento</title> <!-- Titulo de la pagina -->
  <link rel="icon" href="img/icon.png"> <!-- Icono de la pagina -->
  <!-- [ Metadatos ] -->
  <meta charset="UTF-8"> <!-- Codificacion -->
  <meta name="author" content="RukkaPlus"> <!-- Autor -->
  <meta name="description" content="Esta es una pagina de prueba"> <!-- Descripcion -->
  <meta name="keywords" content="pruebas, tests, prueba, test, sandbox"> <!-- Tags -->
  <!-- <meta name="robots" content="noindex"> Hace a la pagina no indexable -->

  <link rel="stylesheet" href="css/style.css"> <!-- Importando CSS -->
  <script src="js/script.js"></script>   <!-- Importando JavaScript -->
</head> <!-- Final de la etiqueta -->
</html>

Etiqueta body

La etiqueta de cuerpo body es importante porque compone la parte visible y la estructura mas solida de una pagina Web. La etiqueta body junto con su contenido sera renderizada en el navegador del cliente

<!DOCTYPE html>
<html>
<body> <!-- Inicio de la etiqueta -->
  <h1>Hello, world!</h1>
</body> <!-- Final de la etiqueta -->
</html>

CSS y Javascript

CSS es un lenguaje Web que permite manejar la estetica de una pagina Web. Para trabajar con CSS se utiliza la etiqueta style o link

<link rel="stylesheet" href="direccion/del/archivo.css">
<style>
  /* Code here... */
</style>

JavaScript es un lenguaje Web multiparadigma y de tipado debil que permite manejar el comportamiento de una pagina Web. Para trabajar con JavaScript se utiliza la etiqueta script

<script src="direccion/del/archivo.js"></script>
<script>
  // Code here...
</script>
