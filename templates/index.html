Python

Python es un lenguaje de programacion multiproposito de ultima generacion desarrollado por Guido van Rossum como substituto de ABC lang en los anios 80. Se trata de un lenguaje de programacion multiplataforma e interpretado de codigo abierto
Python es sin dudas uno de los mas famosos lenguajes de programacion del mundo. Y sirve para practicamente cualquier cosa que puedas imaginarte. Ademas de ser un lenguaje de programacion bastante sencillo de aprender y utilizar

wxPython

wxPython es una libreria popular y muy completa que permite el desarrollo de interfaces graficas de usuario (GUI) nativas de escritorio (Windows, Linux y MacOS). Tiene una tonelada de widgets y es muy utilizado profesionalmente

Flask

Flask es un micro-framework orientado al desarrollo Web (Backend). Es sencillo y rapido de usar; desarrollado para ser extensible y de rapido desarrollo, entrega un entorno bastante minimalista y sencillo

Django

Django es un framework orientado al desarrollo Web (Backend) completo, seguro y profesional. Con la filosofia de baterias incluidas, entrega un entorno muy completo y estable para trabajar del lado del servidor
